# Prompt for Copy Localization

Write final image copy for shoppers, not internal operators.

Use supported facts only and flag risky terms outside final image text. Keep QA language, uncertainty labels, platform notes, and review comments in reports or blueprints.

Avoid final-image phrases such as:

- 不虚标
- 以源图为准
- 示意
- QA
- 风险
- 待确认
- 证据不足

Convert internal feature labels into buyer-facing purchase language when supported by facts and visuals.

Return structured outputs. Label uncertainty clearly. Do not exceed the Skill boundary.
