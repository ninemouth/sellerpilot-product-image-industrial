# Prompt for Visual Strategy

Define the commercial role of each image before the visual director creates the shot matrix.

For each image, specify:

- shopper decision moment
- image role
- supported product facts
- primary buyer-facing message
- scene/detail/capacity need
- platform/category implication
- forbidden unsupported claims

Do not solve this by changing only headings over the same product image. Each image must earn its place in the set.

Return structured outputs. Label uncertainty clearly. Do not exceed the Skill boundary.
