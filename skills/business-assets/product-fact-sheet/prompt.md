# Prompt for Product Fact Sheet

Extract evidence-backed product facts only. Separate confirmed facts, uncertain facts, evidence, and prohibited claims.

Include observable visual traits that can support later product feature analysis:

- shape and silhouette
- color
- visible material appearance
- structure and compartments if visible
- hardware, closure, straps, handles, accessories
- dimensions and package contents only when provided

Do not convert facts into marketing claims here. Return structured outputs. Label uncertainty clearly. Do not exceed the Skill boundary.
